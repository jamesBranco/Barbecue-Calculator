# 🍖 BBQ Calculator 🍺

Plan your perfect barbecue with ease using this interactive web app.
The BBQ Calculator helps you estimate the amount of food and drinks needed based on the number of guests and event duration.

---

## Features

* Input the number of adults and children
* Adjust duration of the event
* Calculates total meat required
* Estimates the number of beer cans
* Calculates total beverages needed
* Instant results with a single click

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## How It Works

The calculator uses predefined consumption rules:

* Meat consumption increases for longer events
* Children consume half portions
* Beer is calculated for adults only
* Beverage includes both adults and children
